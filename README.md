# Supply Chain Correlation Matrix Analysis

This repository contains:
- `correlation.csv` containing the full correlation matrix
- `heatmap.png` showing the Excel conditional formatting (Red → White → Green)

Files were generated following Excel guidelines for supply chain analytics.

## Metrics Covered
Lead time, inventory levels, order frequency, delivery performance, and cost per unit.

## Outcome
Quickly identifies positive/negative relationships between supply chain KPIs to support operational decision-making.
